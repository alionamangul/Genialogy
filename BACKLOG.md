### 🗑️ Удаление фото/документов — ✅ сделано 21.06.2026

Реализовано на VPS-версии (`serve_server.py` + `index.html`):
- Кнопка ✕ на каждой миниатюре фото (боковая карточка + вкладка «Медиа») и на документах; на тач-устройствах видна всегда, на десктопе — по наведению.
- Подтверждение перед удалением; из UI убирается только после ответа сервера.
- Эндпоинты `POST /api/delete-photo` и `/api/delete-doc` (personId + ref): убирают ссылку из `data.json` и удаляют файл с диска.
- **Защита общего файла:** физический файл удаляется только если на него больше никто не ссылается (проверено тестом).
- Работает со всеми видами ссылок (голый id → `.jpg`, имя файла с расширением, внешний URL — у последнего файл не трогаем). Битое/отсутствующее фото показывается плейсхолдером, чтобы «висячую» ссылку тоже можно было удалить.
- Проверено end-to-end локально и на проде (загрузка→удаление, файл исчезает, данные обновляются).
