# 🌳 Генеалогическое дерево

Интерактивный семейный сайт-родословная для двух родов: **Шахназарян** (армянская ветвь)
и **Курудимовы / Мангул** (белорусская и украинская ветвь).

🔗 **Сайт:** https://genialogy.vercel.app

## Что умеет

- 🌳 Интерактивное дерево с фото-аватарками, масштабом и перетаскиванием
- 📇 Список всех людей с поиском
- 🖼️ Галереи фотографий и документов с загрузкой прямо с телефона
- 📖 «Книга» — семейная летопись с перелистыванием страниц (адаптирована под мобильные)
- 🗺️ История, факты и география рода
- ✏️ Редактирование карточек людей прямо на сайте

В базе **295 человек** и **120 семей**.

## Технологии

- **Фронтенд:** один файл `index.html` (HTML + CSS + JS, без сборки)
- **Данные:** `data.json` в репозитории (источник истины, версионируется через git)
- **Медиа:** Vercel Blob (CDN для фото и документов)
- **Бэкенд:** Python serverless-функции на Vercel (`api/*.py`)
- **Деплой:** Vercel, автоматически при push в `main`

## Запуск локально

```bash
python3 serve.py
# открыть http://localhost:8081/index.html
```

> После правки `index.html` или `serve.py` скопируй файл в `/tmp/family-tree/`, иначе
> браузер покажет старую версию. Подробности — в [CONTEXT.md](CONTEXT.md).
